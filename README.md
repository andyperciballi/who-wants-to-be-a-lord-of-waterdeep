# who-wants-to-be-a-lord-of-waterdeep

a D&amp;D (5e 2024) Trivia game where players can progress through a series of multiple choice questions and win!

As a player, I want to have a start button and see instructions for the trivia game so I can test my D&D knowledge.
As a player, I want the theme, wording, and UI to feel like Dungeons & Dragons so playing feels immersive.
As a player, I want to be presented with a clear question with multiple-choice answers so I can choose one confidently.
As a player, I want my progress (question number out of 8) shown so I know how far I’ve advanced in the challenge. this could be displayed like in the game show with a Gold Piece figure
As a player, I want the theme, wording, and UI to feel like Dungeons & Dragons so playing feels immersive.
As a player, I want immediate feedback when I choose an answer so I know whether I was correct or not.
As a player, I want my remaining “lives” (3 fails) displayed so I can track how close I am to losing.
As a player, I want to see a loss screen if I reach 3 wrong answers so I understand the game is over. with a restart button
As a player, I want to see a win screen if I correctly answer 5/8 questions so I feel rewarded. with a restart button
As a player, I want simple instructions at the start so I know how the game works (8 to win, 3 to lose).
